## how to run?

### clone
```
$ git clone https://github.com/wonwoo/frontend-fake-api.git

```
### run
```
$ mvn spring-boot:run
```
or
```
$ mvn install
$ java -jar target/frontend-fake-api-0.0.1-SNAPSHOT.jar
```
